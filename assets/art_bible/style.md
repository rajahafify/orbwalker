# Orbwalker Production Art Bible

Status: draft for assetgen workflow
Source inventory: assets.json

## Intent

Orbwalker is a fantasy match-3 roguelike with mobile-first readability. Production art should make the core loop legible at a glance: elemental orb matching, enemy intent, hero survival, shop choices, treasure chest rewards, and run results.

## Visual Direction

- Genre: stylized dark-fantasy adventure, not casino or abstract puzzle UI.
- Mood: readable, tense, heroic, warm gold against cool dungeon shadows.
- Shape language: sturdy stone, beveled dark metal, carved trim, jewel orbs, clear silhouettes.
- Rendering: polished 2D game illustration with crisp edges and strong value separation.
- Avoid: gore, horror excess, modern objects, casino motifs, illegible painterly blur, watermark-like marks, unapproved embedded text.

## Palette

- Base: charcoal, blackened steel, muted slate, deep stone.
- Accent: antique gold, warm amber, brass trim.
- Element colors: fire ruby/orange, ice cyan/blue, earth moss/stone green, heart rose, armor silver, gold rich yellow.
- UI states must remain distinguishable for affordable, unaffordable, selected, sold out, disabled, warning, victory, and defeat.

## Readability Rules

- Mobile portrait readability is the primary target.
- Icons must read at shop/loadout scale before decorative detail is considered successful.
- Character portraits need clear face/body silhouette and faction identity.
- VFX must communicate source, target, and effect without covering the board.
- Marketing art may be richer, but must still show the actual game fantasy: hero, orbs, dungeon, treasure, combat/shop loop.

## Transparency Rules

Transparent assets require real alpha. No baked checkerboard, matte halos, color spill, jagged alpha, or unapproved baked drop shadows. Test on white, black, gray, checkerboard, and representative in-game backgrounds before approval.

## Production Gate

No generated asset is production-ready until QA, human review, license/source review, metadata completion, and integration approval are complete.
