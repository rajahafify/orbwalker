Combat VFX Quality Settings

Documented May 23, 2026. This captures the first split between low and high post-match combat VFX modes.

Current Default

Low is now the default post-match combat VFX quality.

  • low skips the Max combat VFX overlay and uses the older lightweight generated/SVG-style fallback effects.
  • high keeps the current Max overlay path with layered imported assets, atmospheric sheets, particles, and impact recipes.
  • The presenter still accepts a project setting override at matchatro/combat/post_match_vfx_quality.

Implementation Notes

  • CombatVfxPresenter owns the quality normalization and exposes set_post_match_vfx_quality(), post_match_vfx_quality(), and post_match_vfx_quality_options().
  • The Max overlay checks now require High quality before delegating to CombatMaxVfxOverlay.
  • The existing fallback path was preserved, so Low mode does not need a separate VFX implementation yet.
  • The VFX gallery show page now includes a Low/High selector and defaults to Low for comparison work.
