Wave 9 guardrail: tools/quality/owner_reach_in_budget.gd now ratchets the reach-in table above in CI. A PR fails if it adds a new owner-private reach-in file, increases any baseline, or lowers a count without lowering the checked-in baseline in the same change. This makes relocation visible mechanically instead of relying on review memory.
