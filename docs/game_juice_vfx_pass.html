Game Juice VFX Pass

Documented June 2, 2026. This pass focuses on making default combat feedback readable and tactile on mobile.

Combat Feedback

  • Low-tier post-match VFX was raised so the default setting now has larger impacts, denser sparks, brighter flashes, stronger result labels, and baseline impact rings.
  • Combat hit feedback now supports screen nudge and hit-stop for damage and enemy impacts while respecting reduced motion.
  • Match clears now send an elemental-colored mastery fill stream from the matched board group into the corresponding mastery card.
  • The mastery fill stream uses generated rays, glows, sparks, and a landing halo so it remains available in the trimmed Android build.

Motion And Settings

  • Orb gravity and refill overlays now use drop motion with squash/stretch and overshoot before settling into the board.
  • Combat settings now expose VFX quality and reduced motion, persisted through RunState.
  • Reduced motion keeps feedback visible but removes screen nudge, hit-stop, long spark bursts, and traveling mastery-fill streams.

Android Export

  • The Android preset excludes unused raw/source art packs and compresses native libraries.
  • The APK size was reduced from roughly 622 MB to roughly 195 MB while preserving lightweight runtime VFX.
  • High/max imported VFX packs are excluded from Android; default low/runtime effects remain available.

Validation

  • Full Godot test suite passed after the VFX, settings, motion, and export changes.
  • git diff --check passed.
  • Android debug APK was rebuilt and installed successfully over USB to the OnePlus device.
