Short Description

Orbwalker is a portrait-first match-3 roguelite demo where every orb match powers combat, and every shop choice changes how your next combo hits.

Page Body

Orbwalker is an alpha/demo build of a dungeon-crawling match-3 roguelite. Drag across the board, line up elemental matches, and turn combos into damage, shields, healing, and gold. Between fights, spend your gold in a small shop to buy equipment, relics, mastery upgrades, and treasure chests. Your build changes how each orb type performs, so a simple Fire match can become a bigger hit once your Attack and Mastery grow. This demo focuses on the first playable run loop: main menu, tutorial, combat, shop, boss rewards, victory/defeat flow, and Android portrait play.

Controls

Android

  • Drag from an orb to move it across adjacent cells.
  • Release to resolve matches and combos.
  • Tap shop cards and buttons to buy, reroll, or continue.
  • Tap highlighted tutorial targets when guidance is active.

Desktop/Editor

  • Mouse drag works like touch drag.
  • Click shop cards and buttons.
  • Portrait layout is the intended demo format.

Download Notes

  • Build file: Orbwalker.apk
  • HTML build file: Orbwalker-web-html-0.1.0-demo.zip, containing index.html and the required Godot Web files.
  • Platform: Android, portrait orientation.
  • Browser play is experimental for this demo; Android remains the primary tested target.
  • Package id: com.artifisi.orbwalker
  • Version: 0.1.0-demo / version code 2
  • Install by downloading the APK to an Android device and accepting Android's sideload/install prompt.
  • For browser play on itch.io, upload the HTML zip and set it as playable in browser.

Known Limitations

  • This is an alpha/demo, not a final balanced release.
  • Some progression, collection, settings, and achievement surfaces are present but not final.
  • The current run is tuned as a compact prototype loop, so content variety is intentionally limited.
  • Android CLI export can leave the desktop export process open after writing a valid APK; this is a tooling issue and does not affect the installed demo.
