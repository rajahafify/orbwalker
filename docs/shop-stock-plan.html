Shop Stock Readability Plan

Goal: make the stock cards, dungeon relic, and shop action row feel like the reference: larger art, stronger rarity identity, framed price badges, and action buttons that read as big fantasy controls.

Priority List

P0 - Structure And Scale

  • Refactor the yellow/red section into clear parts: stock-section, offer-grid, offer-card, relic-banner, and shop-action-row.
  • Increase stock card height usage: rarity tab, large item name, large art well, compact effect copy, and framed price button.
  • Move away from current empty bordered rectangles and toward full-card rarity backgrounds.

P1 - Offer Cards

  • Use rarity-colored frames and title colors: epic purple, rare red, uncommon blue/green.
  • Make art the visual center of each card; generated or derived item art should be much larger than current icons.
  • Replace tiny state text with a large framed price badge. Disabled states should dim the whole card and show a readable badge.

P1 - Dungeon Relic

  • Turn the relic into a horizontal banner below stock cards.
  • Use big relic art on the left, name/tier/effect in the middle, and a large price badge on the right.
  • Keep sold out / not enough gold readable without turning the banner into two tiny warning buttons.

P2 - Action Row

  • Make reroll and continue equal-width framed fantasy buttons.
  • Put cost as a secondary line with a coin icon treatment, not plain inline text.
  • Remove the normal sell action from this row; selling should stay in the slot popover/help flow.

P3 - Enforcement

  • Add layout probe checks for card height, art prominence, relic banner placement, price badge size, and action button touch size.
  • Prefer component helpers in shop_view.gd so future stock/relic edits do not recreate private one-off layouts.

Implementation Plan

Step 1 - Extract Layout Constants

  • Add named rects for stock title, offer card internals, relic banner internals, and action buttons.
  • Update shop_layout_probe_snapshot() with these rects before changing visuals.

Step 2 - Rebuild Cards

  • Replace current offer card drawing with a helper that builds rarity tab, large name, art stage, copy, and price badge.
  • Use rarity style data instead of scattered color logic.

Step 3 - Rebuild Relic And Actions

  • Convert relic to a banner layout with large art and large price/state area.
  • Make reroll and continue buttons equal visual weight and remove the visible sell tip from the normal row.

Step 4 - Verify

  • Run focused shop layout tests, git diff --check, and headless res://scenes/shop.tscn --quit-after 2.
  • Visually compare against this mockup and the reference image before committing.
